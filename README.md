### Description
Task Manager is a Django-based web application for managing tasks.
Users can register, create tasks, assign executors, change task statuses, and manage labels.

## Installation:

```bash
git clone git@github.com:BuilovAlmaty/python-project-52.git
cd task_manager
make install
```

### Enviroment variables

Create .env file in the project's root:
```bash
SECRET_KEY=your_secret_key
```

### Start in devmode
```bash
make dev
```
